01 Initiate & Track

Workflows can be initited by investors, who have acces to this through their own digital interface. As soon as a workflow is initiated, the Workflow Manager is able to start tracking it, and alerting everyone else that a new workflow has been started.

Configuration

Configuration infrastructure controls what workflows can be started by each persona. It also provides limits specific to each client, based on their own rules for what users are able to do.

Content

The same workflow will present different content, depending on which persona is initiating the work. Content management also provides help and how-to information.

Machine Learning / AI

In this phase of the workflow, predictions about the most meaningful actions dive suggestions about workflows to initiate.

Usage & Practice Analytics

Initiation is also the trigger to start logging usage data about the workflow. This will be used in the Tracking interface, and also drive recommendations about making the practice more efficient.
